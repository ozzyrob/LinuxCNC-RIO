# Plugin: joint_rcservo

RCSERVO Joint Output

rate in Hz (default: 50Hz)

center in ms (default: 1.5ms)

range in ms (default: +-0.5ms)


```
{
    "type": "joint_rcservo",
    "cl": false,
    "rate": 100,
    "center": 1.5,
    "range": 0.5,
    "pins": {
        "pwm": "C15"
    }
},
```
