# Plugin: vout_7seg

to display a value on a 7seg Display

## example config

```
{
    "type": "vout_7seg",
    "pins": {
        "en1": "PIN_137",
        "en2": "PIN_135",
        "en3": "PIN_136",
        "en4": "PIN_133",
        "seg_a": "PIN_143",
        "seg_b": "PIN_144",
        "seg_c": "PIN_1",
        "seg_d": "PIN_141",
        "seg_e": "PIN_142",
        "seg_f": "PIN_138",
        "seg_g": "PIN_2"
    }
}
```
