# Plugin: joint_pwmdir

PWM Joint Output with DIR-Pin

```
{
    "type": "joint_pwmdir",
    "cl": false,
    "pins": {
        "pwm": "R14",
        "dir": "T14",
        "en": "D16"
    }
}
```
